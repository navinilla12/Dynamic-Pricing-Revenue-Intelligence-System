# Dynamic Pricing & Revenue Intelligence System

## Overview
This project simulates a hospitality revenue management system using Airbnb dataset. It predicts optimal pricing using machine learning and demand-based signals.

## Features
- Price prediction using ML
- Demand scoring model
- Revenue optimization simulation
- Outlier filtering & feature engineering

## Tech Stack
Python, Pandas, Scikit-learn, RandomForest

## How it works
1. Load Airbnb dataset
2. Clean and preprocess data
3. Train ML model for price prediction
4. Generate optimized pricing using demand signals

## Output
- Predicted price vs optimized price
- Revenue intelligence signals

## Use Case
Hospitality pricing optimization, hotel revenue management simulation
